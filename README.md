# Kalkulator Right Issue IDX

Kalkulator HTML mandiri untuk menghitung dampak right issue atau HMETD di Bursa Efek Indonesia.

## Fitur

- Harga teoritis ex-right atau TERP.
- Nilai teoritis 1 HMETD.
- Persentase dilusi jika tidak menebus.
- Jumlah HMETD berdasarkan lot yang dimiliki.
- Perbandingan skenario: tebus, jual HMETD, atau diamkan.
- Tab edukasi singkat untuk pemula.
- Self-check lokal terhadap 5 kasus resmi BEI.

## Cara Pakai

Buka `index.html` langsung di browser. Tidak perlu server, build step, CDN, API key, atau koneksi internet.

## Sumber Riset

Ringkasan rumus dan verifikasi kasus ada di `riset-mekanisme.md`.

## Disclaimer

Alat ini untuk edukasi dan pemeriksaan hitungan. Ini bukan rekomendasi beli, jual, atau tebus efek apa pun. Harga pasar HMETD riil bisa berbeda dari nilai teoritis.
